开发文档：吹牛酒肆 —— 一款4人联机吹牛 bluff 派对小游戏
1. 游戏概述
创意名称：吹牛酒肆
主题：宋代江湖酒肆（古风水墨+赛博酒馆混搭）。四位素不相识的“江湖酒客”（玩家）在荒野路边酒肆里借酒消愁，玩起最刺激的“吹牛”游戏。谁敢吹、谁会骗、谁能笑到最后，谁就是“酒霸”！
核心卖点（适合中国人玩）：

100%还原“吹牛”文化（中国人从小玩的“大话骰子”），结合经典酒桌骗局。
古风沉浸感：水墨UI、古琴BGM、诗词吐槽（如“酒逢知己千杯少，骗子相逢一杯毒”）。
短平快：一局5-10分钟，完美适合微信群约战、宿舍联机。
未来目标：微信小程序（纯web版先上线，方便分享链接秒进）。
玩家规模：固定4人联机（支持房主创建私密房间+分享链接/二维码）。

胜利条件：活到最后（每人2条命，输两次淘汰）。
平台：暂时纯Web（HTML5），后续用Taro/uni-app一键转微信小程序。
2. 游戏机制与玩法
游戏保留《骗子酒吧》核心乐趣（吹牛+惩罚+心理博弈），但全部本土化、古风化。支持两种模式，房间创建时可选（或轮换玩）。
2.1 骰子模式（经典“吹牛骰”升级版）

准备：每人1个“酒盅”+5颗骰子（虚拟，服务端摇）。
流程（逆时针）：
所有人同时摇骰（仅自己可见，可点“查看”）。
第一人喊“总共有X个Y点”（Y=1-6，X≥1）。
下家必须加码（X增大或Y任意但X≥当前）或质疑（喊“骗子！”）。
质疑后全员亮骰盅：实际数量≥喊的值 → 质疑者输；反之 → 喊者输。

惩罚：输家“饮下一杯蒙汗药”（喝1瓶毒酒）。每人最多2瓶，2瓶=淘汰（画面直接“醉倒”）。
回合结束：淘汰一人后重新摇骰，继续直到剩1人。

中国风创新：骰子点数可显示“梅兰竹菊”图案；喊话支持快捷诗词（如“一壶浊酒喜相逢，三杯两盏下肚骗”）。
2.2 扑克模式（吹牛牌升级版）

准备：20张“酒令牌”（6张“花雕”、6张“女儿红”、6张“竹叶青”+2张“赖子/蒙汗药”）。每轮随机指定1种为主牌，每人发5张。
流程（逆时针）：
第一人出1-3张牌（背面朝上），必须喊“X张主牌”（可全撒谎）。
下家可跟牌（也出1-3张主牌）或质疑（翻开上家牌）。
质疑成功（上家撒谎）→ 上家输；质疑失败（上家真）→ 下家输。
没人质疑则继续，直到有人出完手牌或强制质疑。

惩罚：输家“转动左轮酒壶”（俄罗斯轮盘模拟）：6格酒壶，1格藏“真蒙汗药”。每输一次转1格（位置固定，累积推进），中毒=喝1瓶（2瓶淘汰）。
赖子：可当任意主牌，增加吹牛空间。

中国风创新：牌面是古风酒名+水墨插画；质疑成功时弹出“开他！”动画+古筝音效。
通用规则：

上一轮输家先手。
支持实时语音聊天（WebRTC）+表情包（“我信了你的邪”“纯纯骗子”）。
游戏时长短，容错高，新手10秒上手。

3. 联机实现

实时同步：所有游戏状态（手牌、骰子结果、喊话、质疑）通过WebSocket实时推送。
防作弊：
骰子/发牌全部服务端生成（使用Crypto.random），客户端只收到自己的数据。
质疑验证在服务端执行，防止客户端修改。
房间锁：4人满员后自动开始，掉线自动补AI（简单规则AI）。

房间系统：
房主创建房间 → 生成6位房间码/分享链接/小程序二维码。
支持“好友对战”或“快速匹配”（可选）。
断线重连：5秒内自动恢复状态。


技术选型（纯Web友好，后续小程序无缝迁移）：

前端：Vue3 + Vite + Canvas（或Phaser.js轻量2D渲染酒桌）。
实时：Socket.io（Node.js）。
语音：WebRTC（PeerJS简化）。
部署：Vercel / Railway（免费）。

4. 代码路线（Roadmap）
阶段1（MVP，1-2周）：

后端：Node.js + Express + Socket.io
/rooms 创建房间、加入、状态同步。
Game Engine：类DiceGame、CardGame，状态机管理（waiting / shaking / calling / challenging / punishment）。

前端：Vue3组件
Lobby.vue（房间列表）
Table.vue（酒桌主界面：4个座位+酒盅/牌堆+聊天栏）
AnimationLayer（蒙汗药动画、转酒壶粒子特效）。

测试：本地4个浏览器标签模拟4人。

阶段2（打磨，1周）：

加入两种模式切换、AI、历史战绩。
优化移动端适配（触屏摇骰子手势）。
音效+BGM（免费古风素材）。

阶段3（小程序迁移，1周）：

用Taro 3 重构前端 → 微信小程序原生Canvas+WebSocket。
接入微信群分享、好友邀请、成就系统。

5. 创新点（区别于原版，突出中国风+小程序优势）

文化本土化：把“骗子酒吧”彻底中国化——“蒙汗药”取代毒药/子弹，“吹牛”取代英文叫牌，古风水墨UI+诗词吐槽，零文化隔阂，中国玩家秒懂秒爱。
视觉沉浸：酒桌第一人称视角+动态烛光+酒客表情（吹牛时“奸笑”动画），小程序也能流畅。
社交放大：内置“开麦喷人”+表情快捷键，完美复刻酒桌真实体验；支持“录像回放”分享朋友圈（小程序独有）。
轻量高可玩：纯Web零安装，5分钟上手；两种模式随机切换，防腻；未来小程序可加“每日酒令挑战”单机模式。
技术创新：服务端验证+客户端渲染结合，保证公平又丝滑；惩罚动画用Canvas粒子+SVG，视觉冲击强于原版。
商业潜力：小程序内购皮肤（不同朝代酒客头像、古风牌背），或广告变现，轻松变现。

总结：
《吹牛酒肆》是一款“酒桌文化+现代联机”的完美融合体，既保留了《骗子酒吧》的核心爽感，又注入浓浓中国味，适合微信生态传播。MVP可在一月内上线纯Web版，之后无缝转小程序，轻松冲上小游戏榜单。
