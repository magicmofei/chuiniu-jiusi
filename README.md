# 🍶 吹牛酒肆 · Chuiniu Jiusui

> 一款4人联机古风吹牛骰子小游戏

## 技术栈

| 端 | 技术 |
|---|---|
| Backend | Node.js + TypeScript + Express + Socket.io |
| Frontend | Vue 3 + Vite + TypeScript + Pinia + Vue Router + TailwindCSS |

## 快速启动

### 方式一：根目录一键启动（推荐）

```bash
# 安装根依赖
npm install

# 同时启动前后端
npm run dev
```

### 方式二：分别启动

**后端**
```bash
cd backend
npm install
npm run dev
```

**前端**
```bash
cd frontend
npm install
npm run dev
```

## 端口

- Backend: http://localhost:3000
- Frontend: http://localhost:5173
