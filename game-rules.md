# 吹牛酒肆 - 核心规则（复刻版）

## 基础设置

| 项目 | 说明 |
|------|------|
| 人数 | 2-4 人 |
| 牌组 | Q×6 + K×6 + A×6 + Joker×2 = 20 张 |
| 手牌 | 每人每局发 **5 张** |
| 生命 | 每人 **6 瓶酒**，其中 **1 瓶**是蒙汗药（随机位置） |

## 回合流程

### 第一阶段：设定目标
- 系统洗牌，每人发 5 张
- 系统**随机**指定本局「目标牌」（Q / K / A 三选一，公开显示）

### 第二阶段：轮流出牌
- 玩家按顺时针轮流操作
- 每次选 **1-3 张**牌扣下（面朝下），声称「全是目标牌」
- 可以撒谎（混入非目标牌），也可以诚实出牌
- 下一位玩家可选择：继续出牌 **或** 质疑上家

### 第三阶段：质疑结算
- 任意玩家喊「你在撒谎！」即触发质疑
- **翻开**上一位玩家最后打出的牌
- 判定规则：只要其中有**任意一张**既不是目标牌、也不是 Joker → **判定撒谎**
- Joker 是万能牌，可充当任意牌

| 质疑结果 | 受罚方 |
|----------|--------|
| 质疑成功（出牌者确实撒谎） | 出牌者喝酒 |
| 质疑失败（出牌者没撒谎） | 质疑者喝酒 |

### 第四阶段：饮酒惩罚
- 输家从自己面前的酒瓶中**随机选一瓶**喝下
- 喝到**普通酒**：存活，进入下一局
- 喝到**蒙汗药**：该玩家**出局**
- 无论结果，喝完后**清空桌面，重新洗牌发牌**

## 胜利条件

- **最后幸存者**获胜
- 当其他所有玩家都因喝到蒙汗药倒下时，最后一人赢得「酒霸」称号

## 关键规则速查

| 动作 | 规则 |
|------|------|
| 出牌 | 选 1-3 张扣下，声称全是目标牌 |
| 质疑 | 有任意一张不是目标牌且不是 Joker → 撒谎 |
| 饮酒 | 从剩余 6 瓶中随机选 1 瓶，中毒即出局 |
| 洗牌 | 每次质疑结算（饮酒）后，立即洗牌重发 |
